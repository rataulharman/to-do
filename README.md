# React To-Do App

This is a simple To-Do List application built using React as part of a college assignment.  
The app uses the `useReducer` hook for state management.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Clean and responsive dark UI design

## Technologies Used

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- useReducer Hook
- FontAwesome Icons

## State Management

This project uses the `useReducer` hook to manage all task operations:

- ADD_TASK
- EDIT_TASK
- DELETE_TASK
- COMPLETE_TASK
